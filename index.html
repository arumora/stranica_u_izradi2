Moje vještine

AutoCAD

U dosadašnjem poslu sam puno koristila program Autocad. Ovaj program omogućuje izradu geodetskih elaborata, kako bi što bolje prikazali podatke dobivene na terenu i predočili ih investitorima elaborata.

Q gis

Završila sam osnovni tečaj za QGIS i koristila ga na studiju Geoinformatike. Ovaj program omogućuje mnoge prostorne analize koje mogu biti jako korisne klijentima!

Web development

Završila sam Web development tečaj na online platformi Smart Njinja, koji me naučio osnovima izrade web stranica - HTML-a i CSS-a. Educiram se i dalje kako bi napredovala u tome znanju, da web stranice za klijente izgledaju što bolje!
